Базовые API 1C (http://localhost:5000/api/ONEC/)
proxy POST/GET

Обратиться к серверу 1С с POST/GET запросом

options POST/GET

Сохранить\Получить параметры связи с сервером 1С

startonecsession \ stoponecsession GET

Запустить\Остановить связь с сервером 1С

onecsessionstatus GET

Получить статус связи с сервером 1С

start \ stop GET
